API для Yatube
Yatube API — REST API для социальной сети Yatube, разработанный на Django REST Framework.

Проект предоставляет программный интерфейс для работы с публикациями, комментариями, сообществами и подписками пользователей. 
Авторизация реализована с использованием JWT-токенов.

API позволяет сторонним клиентам (веб-приложениям, мобильным приложениям и другим сервисам) взаимодействовать с платформой Yatube без использования серверных HTML-страниц.

Функциональные возможности:

Публикации
1. просмотр списка публикаций;
2. получение отдельной публикации;
3. создание публикаций;
4. редактирование собственных публикаций;
5. удаление собственных публикаций;
6. поддержка пагинации.

Комментарии
1. просмотр комментариев публикации;
2. создание комментариев;
3. редактирование собственных комментариев;
4. удаление собственных комментариев.

Сообщества
1. получение списка сообществ;
2. просмотр информации о конкретном сообществе.

Подписки
1. подписка на авторов;
2. просмотр собственных подписок;
3. поиск по подпискам;
4. защита от повторных подписок;
5. запрет подписки на самого себя.

Авторизация
1. получение JWT-токена;
2. обновление JWT-токена;
3. проверка JWT-токена.

Технологии:

Django

Установка проекта
Клонировать репозиторий:
git clone https://github.com/<your_username>/api-final-yatube.git

Перейти в директорию проекта:
cd api-final-yatube

Создать виртуальное окружение:
python -m venv venv

Активировать виртуальное окружение:
Windows:
venv\Scripts\activate

Linux/macOS:
source venv/bin/activate

Установить зависимости:
pip install -r requirements.txt

Перейти в папку проекта:
cd yatube_api

Выполнить миграции:
python manage.py migrate

Запустить сервер:
python manage.py runserver

Документация API
После запуска проекта документация доступна по адресу:
http://127.0.0.1:8000/redoc/

Аутентификация
Получение JWT-токена:
POST /api/v1/jwt/create/

Права доступа
Неавторизованные пользователи имеют доступ только на чтение.

Авторизованные пользователи могут:
1.создавать публикации;
2. создавать комментарии;
3. оформлять подписки.

Изменять и удалять публикации и комментарии может только их автор.
