API для Yatube
Описание проекта

Yatube API — REST API для социальной сети Yatube, разработанный на Django REST Framework.

Проект предоставляет программный интерфейс для работы с публикациями, комментариями, сообществами и подписками пользователей. Авторизация реализована с использованием JWT-токенов.

API позволяет сторонним клиентам (веб-приложениям, мобильным приложениям и другим сервисам) взаимодействовать с платформой Yatube без использования серверных HTML-страниц.

Функциональные возможности
Публикации
просмотр списка публикаций;
получение отдельной публикации;
создание публикаций;
редактирование собственных публикаций;
удаление собственных публикаций;
поддержка пагинации.
Комментарии
просмотр комментариев публикации;
создание комментариев;
редактирование собственных комментариев;
удаление собственных комментариев.
Сообщества
получение списка сообществ;
просмотр информации о конкретном сообществе.
Подписки
подписка на авторов;
просмотр собственных подписок;
поиск по подпискам;
защита от повторных подписок;
запрет подписки на самого себя.
Авторизация
получение JWT-токена;
обновление JWT-токена;
проверка JWT-токена.

Установка проекта

Клонировать репозиторий:

git clone https://github.com/<your_username>/api-final-yatube.git

Перейти в директорию проекта:

cd api-final-yatube

Создать виртуальное окружение:

python -m venv venv

Активировать виртуальное окружение:

Windows:

venv\Scripts\activate

Linux/macOS:

source venv/bin/activate

Установить зависимости:

pip install -r requirements.txt

Перейти в папку проекта:

cd yatube_api

Выполнить миграции:

python manage.py migrate

Запустить сервер:

python manage.py runserver
Документация API

После запуска проекта документация доступна по адресу:

http://127.0.0.1:8000/redoc/
Аутентификация

Получение JWT-токена:

POST /api/v1/jwt/create/

Тело запроса:

{
    "username": "username",
    "password": "password"
}

Пример ответа:

{
    "refresh": "token",
    "access": "token"
}
Примеры запросов
Получить список публикаций
GET /api/v1/posts/

Пример ответа:

[
    {
        "id": 1,
        "text": "Первый пост",
        "author": "lilia",
        "group": 1
    }
]
Создать публикацию
POST /api/v1/posts/

Заголовок:

Authorization: Bearer <access_token>

Тело запроса:

{
    "text": "Мой первый пост",
    "group": 1
}
Подписаться на автора
POST /api/v1/follow/

Заголовок:

Authorization: Bearer <access_token>

Тело запроса:

{
    "following": "author_username"
}
Права доступа

Неавторизованные пользователи имеют доступ только на чтение.

Авторизованные пользователи могут:

создавать публикации;
создавать комментарии;
оформлять подписки.

Изменять и удалять публикации и комментарии может только их автор.
